Para utilizar a calculadora Poli-Calculator deve-se seguir os seguintes cuidados:

Para uso de números negativos, preferencialmente utilziar para o primeiro número da operação.
Para uso do comando de exclusão de itens do registro de memória completa (MC da calculadora) ou itens específicos 
(MC do painel a direita, até os 4 últimos itens) deve-se clicar em AC logo após.
A Calculadora opera com uma operação por vez, então logo após obter o resultado, clique em AC para limpar os dados 
(menos o registro de memória, é claro) e poder realizar a próxima operação.

Para inicializar a calculadora em seu Host Local digite no Prompt de Comando aberto na pasta 'CalculaApp' e digite 
'npm run electron-v'
